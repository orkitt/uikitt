# orkitt_ui

A reusable UI component library for the **Orkitt** ecosystem.

Provides ready-to-use widgets, decorations, extensions, and UI helpers
to build consistent Flutter interfaces faster.

---

## Features

- Buttons, inputs, cards, and loaders
- Reusable UI components
- Decoration & styling helpers
- Widget extensions for layout shortcuts
